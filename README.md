## Features

 - UEFI boot
 - 64-bit x86
 - Colorful graphics
 - PS/2 mouse
 - PS/2 keyboard
 - COM-port debug output


This system is going to be coded exclusively in @hexalang, but I also recommend you to take a look at @ziglang, another promising systems programming language!
